Intel HD Graphics 2000/3000 核显游玩 Minecraft Java 版方法（1.17 之前+1.17 之后）
Nuclear_Creeper lol
  2023-01-19 09:37:01 2023-01-19 09:37  

游玩 1.17 以下

下载此文件并解压（安全无毒，杀毒软件报毒忽略即可，建议加入杀毒软件信任区）

https://www.123pan.com/s/wmalVv-UsYD 提取码：nc66

本来想用 OneDrive 的，结果也报毒给我拦截不让分享了，但是它真的没毒（Microsoft 我谢谢你

![image-20230119094704230](C:\Users\Nuclear Creeper\AppData\Roaming\Typora\typora-user-images\image-20230119094704230.png)

解压后打开 Intel.HD.Graphics.Driver.Win10.32-bit.&.64-bit.Fix.by.MST.exe，提示恶意程序不用管直接运行即可（杀软可能会拦截，记得设置一下）

温馨提示：此程序带有音乐，打开前请注意一下

![image-20230119095054396](C:\Users\Nuclear Creeper\AppData\Roaming\Typora\typora-user-images\image-20230119095054396.png)

打开后先别急着点 Patch，先定位到 C:\Windows\System32 文件夹中找到“ig4icd64.dll”文件，将它剪切到 C:\Windows\SysWOW64 文件夹中，再点击 Patch，Patch 完毕后再剪切回来

然后打开你准备好的 MC 启动器，下载一个 1.17 以下的版本（为了启动速度快可以选择 1.8.9 等较老版本），等待启动完成后游戏窗口出现，你就成功了！

游玩 1.17 及以上

但先别高兴太早，如果你这时下载一个 1.17 或以上的版本启动后会出现这个弹窗：

![image-20230119095847017](C:\Users\Nuclear Creeper\AppData\Roaming\Typora\typora-user-images\image-20230119095847017.png)

（而且离谱的是报错弹窗出来后启动器还会提示游戏启动成功，就离谱好吧

导致这个问题的原因是 Intel HD Graphics 2000 的最新 Windows 驱动最高只支持到 OpenGL 3.1（不知道 OpenGL 是啥的可以自行查一下），而 1.17 及以上渲染核心版本更新至 OpenGL 3.2（或更高），所以弹出这个报错弹窗并无法启动。

然而这样除了换配置你就无缘 MC Java 版1.17 及以上的高版本了吗？

NO NO NO!

首先进入 https://github.com/pal1000/mesa-dist-win/releases 下载 Mesa 的最新构建版本（注意文件名一定要是xxx.release-mingw.7z），如果进不去可以使用 Steam++ 加速 Github 自行使用魔法

或在这里下载（但不能保证是最新版本）

mesa3d-22.3.3-release-mingw.7z 密码：nc66

下载后解压到一个文件夹

![image-20230119100828155](C:\Users\Nuclear Creeper\AppData\Roaming\Typora\typora-user-images\image-20230119100828155.png)

双击 perappdeploy.cmd，先按任意键继续

![image-20230119100928519](C:\Users\Nuclear Creeper\AppData\Roaming\Typora\typora-user-images\image-20230119100928519.png)

![image-20230119100952991](C:\Users\Nuclear Creeper\AppData\Roaming\Typora\typora-user-images\image-20230119100952991.png)

这里需要粘贴你的 Java 的路径（精确到 bin 文件夹，例如我的路径就是 G:\Java\jdk-17.0.5\bin）

![image-20230119101106228](C:\Users\Nuclear Creeper\AppData\Roaming\Typora\typora-user-images\image-20230119101106228.png)

这里输入 java.exe（如果你是用 javaw.exe 启动就输入 javaw.exe）

![image-20230119101200555](C:\Users\Nuclear Creeper\AppData\Roaming\Typora\typora-user-images\image-20230119101200555.png)

输入 2，回车

![image-20230119101230403](C:\Users\Nuclear Creeper\AppData\Roaming\Typora\typora-user-images\image-20230119101230403.png)

输入 y 回车，这是第一次输入 y，算上这一次总共输入两次 y ，后面都输入 n

然后程序自动退出

大功告成！

这是再打开你的启动器，选择一个 1.17 或以上的版本启动游戏（我这里直接用 1.19.3）

（我这里是用了一个第三方作者制作的优化端，原版端也是一样）

![image-20230119101632375](C:\Users\Nuclear Creeper\AppData\Roaming\Typora\typora-user-images\image-20230119101632375.png)

尽情享受高版本独有的乐趣吧~
